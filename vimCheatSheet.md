# vimチートシート

## 移動
### 対応する括弧に飛ぶ
- (CommandMode) %
### 上下

| コマンド | スクロール方向 | スクロール量 | 覚え方  |
|----------|----------------|--------------|---------|
| <C-u>    | 上方向         | 半画面分     | up      |
| <C-d>    | 下方向         | 半画面分     | down    |
| <C-b>    | 上方向         | 一画面分     | back    |
| <C-f>    | 下方向         | 一画面分     | forward |
| <C-e>    | 上方向         | 一行分       |         |
| <C-y>    | 下方向         | 一行分       |         |


## バッファ
- :b <tab>で移動
- :b f　でまえ
- :lsでリスト
- <space> ; でバッファをfzfで開く
## プラグイン
### table mode
- <leader>tm (リーダーはスペースのはず)
- ||入力で区切り線

### fzf
- <space> t ファイル
- <space> r tags
- <space> ; でバッファをfzfで開く
